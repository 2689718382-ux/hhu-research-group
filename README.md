# 河海大学土木与交通学院课题组静态网站

这是一个纯静态课题组网站模板，适合部署到 GitHub Pages、Vercel、Netlify 或 Cloudflare Pages。

## 文件说明

- `index.html`：网站全部内容、样式和脚本都在这个文件中。
- `README.md`：使用说明。

## 本地预览

1. 解压压缩包。
2. 双击 `index.html`，用浏览器打开即可预览。
3. 修改文字后刷新浏览器即可看到效果。

## 建议优先替换的内容

在 `index.html` 中搜索并替换以下内容：

- `河海大学土木与交通学院课题组`
- `面向岩土工程可靠性的智能分析与裂隙结构表征`
- `请替换为论文题目`
- `课题组负责人`
- `example@hhu.edu.cn`
- 论文、项目、成员信息

## 部署建议

最简单方式：上传到 GitHub 仓库，然后在 Settings -> Pages 中启用 GitHub Pages。
